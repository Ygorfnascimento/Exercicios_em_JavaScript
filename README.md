Repositório com **exercícios em JavaScript**, utilizando **HTML** para estruturação, voltados ao desenvolvimento da lógica de programação e à prática de interatividade básica.

---

## 🎯 Objetivo

Aprimorar habilidades em **JavaScript**, praticando:

- manipulação de variáveis  
- estruturas condicionais  
- laços de repetição  
- funções  
- arrays e objetos  
- interação básica com HTML  

---

## 📂 Organização dos exercícios

Os exercícios estão organizados em **arquivos `.js` e `.html`**, cada um abordando um conceito ou conjunto de conceitos.  
Os nomes dos arquivos indicam o tema do exercício, facilitando a navegação e o estudo.

---

## 🛠 Tecnologias utilizadas

- **JavaScript**
- **HTML5**
- **Ambiente de desenvolvimento:** VS Code / Navegador
